# Expense Tracker Dashboard

A full-stack MERN Expense Tracker Dashboard with analytics, transaction management, and interactive UI.

## Features

- Add Income & Expense Transactions
- Delete Transactions
- Real-Time Balance Calculation
- Expense Analytics Pie Chart
- Responsive Dashboard UI
- MongoDB Atlas Integration
- REST API Integration
- Interactive Financial Overview

## Tech Stack

### Frontend
- React.js
- Tailwind CSS
- Axios
- Chart.js

### Backend
- Node.js
- Express.js
- MongoDB Atlas
- Mongoose
